===============================================
    SISTEMA DE CONVERSÃO UTM → GEOGRÁFICAS
===============================================

🚀 COMO EXECUTAR:

1. PRIMEIRA VEZ:
   - Clique duas vezes em: install.bat
   - Aguarde a instalação das dependências

2. EXECUTAR O SISTEMA:
   - Clique duas vezes em: conversao_utm.py
   - OU no terminal: py conversao_utm.py

===============================================

📋 COMO USAR A INTERFACE:

1. SELECIONAR PASTA ORIGEM:
   - Clique em "Selecionar Pasta"
   - Escolha a pasta com seus arquivos CSV

2. ESCOLHER ARQUIVO:
   - Selecione o arquivo CSV na lista
   - O sistema mostrará uma prévia dos dados

3. SELECIONAR COLUNAS:
   - Escolha a coluna X (Easting/Este)
   - Escolha a coluna Y (Northing/Norte)
   - O sistema tenta detectar automaticamente

4. PASTA DESTINO:
   - Clique em "Selecionar Pasta" 
   - Escolha onde salvar o arquivo convertido

5. CONVERTER:
   - Clique em "Converter Arquivo"
   - Aguarde a conversão

===============================================

📊 RESULTADO:

O arquivo será salvo com sufixo "_convertido.csv" 
e terá 4 novas colunas:

- Longitude_DMS: -45° 30' 25.67"
- Latitude_DMS: -23° 15' 42.89"  
- Longitude_Decimal: -45.507130
- Latitude_Decimal: -23.261914

===============================================

⚙️ CONFIGURAÇÃO:

- Sistema: SIRGAS 2000
- Zona UTM: 23S (Hemisfério Sul)
- Meridiano Central: -45°

===============================================

🐛 PROBLEMAS COMUNS:

❌ "Python não encontrado":
   → Use: py conversao_utm.py (em vez de python)

❌ "Dependências não encontradas":
   → Execute: install.bat

❌ "Erro ao carregar CSV":
   → Verifique se o arquivo está em formato CSV válido

❌ "Coordenadas incorretas":
   → Confirme se as colunas X e Y estão corretas
   → Verifique se os dados estão em UTM Zone 23S

===============================================

📁 ARQUIVOS DO SISTEMA:

- conversao_utm.py     ← EXECUTE ESTE
- interface_grafica.py ← Interface
- conversor_utm.py     ← Conversão
- install.bat          ← Instalador
- requirements.txt     ← Dependências
